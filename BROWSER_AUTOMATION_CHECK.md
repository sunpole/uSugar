## Подробности фактической проверки

### Курсор мыши

Проверка изменила координаты курсора и вернула его обратно:

- до: `723,518`;
- после движения: `728,523`;
- после восстановления: `723,518`.

Вывод: управление курсором доступно.

### Окна Windows и Chrome

Chrome был найден как процесс `chrome.exe`. После вызова `SetForegroundWindow` активный заголовок окна стал:

`Telegram Web - Google Chrome`

Вывод: текущая среда может активировать внешнее окно Chrome.

### Вкладки Chrome

Проверка вкладок:

- до переключения: `Telegram Web - Google Chrome`;
- после `Ctrl+Tab`: `Лимиты 5 часов и недели - Google Chrome`;
- после `Ctrl+Shift+Tab`: `Telegram Web - Google Chrome`.

Вывод: текущая среда может переключать вкладки Chrome через keyboard automation.

### Telegram Web

В Telegram Web был выполнен реальный UI-цикл:

1. Активирована вкладка Telegram Web.
2. Курсор поставлен в поле ввода.
3. Напечатано `/version`.
4. Нажат `Enter`.
5. Сделан реальный скриншот экрана.
6. По скриншоту прочитан ответ бота.

Факт ответа:

```text
uSugarBot v1.0.39
Семейный помощник диабета.
Ваш сахар под контролем.
```

Вывод: в текущей среде возможно не только видеть Telegram Web, но и отправлять команды через реальный пользовательский интерфейс.

## Ограничения

- Управление Telegram Web идёт через координаты экрана и keyboard automation, а не через прямой DOM/API-доступ к вкладке.
- Надёжность зависит от того, что Chrome открыт, Telegram Web виден, масштаб/расположение окна не изменились, а поле ввода находится в ожидаемой зоне.
- Чтение ответа выполнялось по реальному скриншоту, а не через программное чтение DOM.
- Для приватных или рискованных действий в Telegram всё равно лучше подтверждение пользователя.

## Итог

Текущая среда Codex может выполнить полный UI-цикл в открытом Telegram Web:

`активация Chrome -> активация Telegram Web -> фокус в поле ввода -> печать команды -> отправка -> скриншот -> чтение ответа -> анализ`

Это значит, что для разработки uSugar можно автоматизировать простые Telegram smoke checks вроде `/version`, `/health`, `/status`, `/help`, если окно Telegram Web открыто и находится в стабильной позиции.
