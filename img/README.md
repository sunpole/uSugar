# OCR Sample Images

This folder contains public uSugar OCR development samples.

The user explicitly provided these screenshots/photos for project development and allowed them to be included in the repository. They are treated as sample inputs for OCR work, not as private records.

## Folders

- `simple/` - early Libre2 screenshots used while building the first Libre2 CV path.
- `simple_new/` - updated Libre2 software screenshots. This set includes both immediate black-and-white scan screens and color/history graph screens.
- `simple_gluk/` - manual glucometer photos for the first `glucometer_photo` OCR branch.

## Safety Rules

- Do not put tokens, `.env`, Telegram IDs, database exports, or ngrok URLs in this folder.
- Do not use these images as proof that OCR is medically reliable.
- Do not auto-save OCR results from these images without explicit user confirmation.
- For public uNews covers, prefer neutral generated/documentation images instead of screenshots with glucose-like values.
